# Integrated Fraud & Credit Risk Prediction Model

## Overview
This project builds:
- Fraud Detection Model (Logistic Regression & Random Forest)
- Credit Risk Model (Upcoming Integration)

## Dataset
Fraud dataset: Download from Kaggle and save as `fraud_raw.csv`
Credit dataset: Save as `credit_raw.csv`

## Installation

Install dependencies:

pip install -r requirements.txt

## Run the project

python fraud_model.py
